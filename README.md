STUDENT GRADE MANAGEMENT SYSTEM

This project is a Python-based desktop application developed as a course project
for SEN 301 (Object Oriented Analysis and Design) at Bells University of Technology.

The system is designed to help manage student academic records by allowing users
to enter student details, calculate Grade Point Average (GPA), and display student
performance using graphical charts.

FEATURES
- Entry and storage of student records
- Automatic GPA calculation
- Visualization of student scores using charts
- Simple and user-friendly interface

TOOLS AND TECHNOLOGIES
- Python programming language
- Tkinter for graphical user interface
- CSV file handling for data storage
- Matplotlib for data visualization

HOW TO RUN THE PROJECT
1. Ensure Python is installed on the system
2. Open the project folder
3. Run the application using:
   python main.py

PROJECT STRUCTURE
- main.py: Main application file
- src/: Contains program modules
- data/: Stores student records
- screenshots/: Contains project screenshots

This project was developed for academic purposes.
